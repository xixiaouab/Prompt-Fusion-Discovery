# Prompt Fusion Discovery Project Page

Static GitHub Pages site for:

**Layer-Specific Prompt Fusion Discovery via Differentiable Search in Vision Foundation Models**

The site follows the VIGIL project-page template and includes:

- title, authors, affiliations, and project links
- teaser figure
- abstract and three main claims
- method overview
- result highlights
- BibTeX block

## Local Preview

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Deployment

Push this repository to GitHub and enable GitHub Pages from the `main` branch root.
